Detailed Description

Classification for LanguageObject instances.

It represents which language element an object belongs to. C++ objects (as well as Python ones) do have a hierarchical structure: Module (the include) | Class | Instance

LanguageType indicates where the object is located in the hierarchy, or a special "None" value (for a "module parent").
